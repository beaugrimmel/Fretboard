Enjoy this Fretboard Memorization Aid!

Too often guitar players don't learn the notes on the neck. Doing this with any other instrument would be ridiculous but for some reason it's common with guitar. This tool can help you practice when you don't have a guitar laying around.

There are two variations to practice (inspired by JustinGuitar).

Game 1 - Identify the Note (Prompts a random fret and string)
Game 2 - Find the Frets (Find all the note locations on the neck)
